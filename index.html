Welcome to Simon Game By Muhammad Nasir Nadeem

Rules are as following:-

  • To start the game, press any key on keyboard.

  • Click the flashed color box for Level 1.

  • For next levels, click the color box of previous level and next levels in Sequence.

  • For example, In Level 1, blue color box flashed, once you click it, you will go to Level 2. In Level 2, red box flashed, now you have to click Blue box then Red box to go to 3rd Level so on and so forth.
